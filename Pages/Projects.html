Fal Tracker

Faltracker is a discord bot that is currently running on a student server for over 300 students in our deparment. Written in python, it scrapes live data from Falmouth Uni’s website to determine:

  • Whether services (shops, cafes, gyms) are open or closed
  • The service's opening times
  • Upcomming Uni held events
  • Lists of helpful uni contact infomation, such as emails and numbers
This data is then formatted depending on the user’s request and then fed back to the user through the correct discord channel.

Its makes use of the Asynchronous HTTP Client/Server for [asyncio](https://docs.aiohttp.org/en/stable/glossary.html#term-asyncio) to extract the html data from a uni webpage. The text in the file is sorted and cleaned for relivance and then the needed data is fed back to the user.

Guitar Controller Game

For my first year module, comp140, I made a game controller out of an old accustic guitar. The pitch is that you can place your finger down on the neck of a guitar, using real wood and real strings, and that would register in a “guitar hero like” game as you playing that certain note.

The uniqueness comes from not having anybuttons but an actual guitar. It was linked through and ardunio conneting to a unity project containing the game.

The guitar controller worked as a large potential divider circuit, where each string was a pin into the ardino and each fret was between a different resistor. A different potential difference would be recorded depending on the fret, which the user played. This would pick up in the game as a note.

In the game, I wrote a mapping system for songs, including relivant song and note infomation, so that you could map out anysong to be played. The project took an OOP approach and ran parallel to the arduni code.
